## 💡 시작하며.. 느낀 점

처음으로 LangChain을 사용해보며, 단순 분류 모델과는 다르게 **정답이 애매하고 사람의 판단이 개입되는 작업**이 많아 흥미로웠습니다. YOLO처럼 명확한 정답이 있는 태스크와는 완전히 다른 접근 방식이라는 점이 인상 깊었고, 이러한 LLM 기술을 활용한 정보 검색 방식(RAG)에 대해 더 심층적으로 이해할 수 있었습니다.
